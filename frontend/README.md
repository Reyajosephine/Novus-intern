# AI SQL Agent Frontend

## Usage

1. Open `index.html` in your browser (or serve via a static server).
2. Ensure the backend FastAPI server is running at `http://localhost:8000`.
3. Enter a natural language question and click "Run Query".

## Features
- Modern, dark-themed dashboard UI
- Natural language to SQL
- Verified, safe, read-only SQL execution
- Dynamic results and verification panel

---

**Note:** For production, serve frontend via a proper static file server and secure backend API access.
